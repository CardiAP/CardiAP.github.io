About CardIAP

Here we present CardIAP, an open-source web application for analyzing series of calcium handling phenomena from confocal microscopy images.

CardIAP is a Python based tool, which allows users to easily work with a single file or a pool of images, and obtain representative amplitude and kinetics data.

Quick start using CardiAP

CardiAP is a Python based tool, which allows users to easily work with a single file or a pool of images, and obtain representative amplitude and kinetics data, and it’s statistical analysis.

The dependences used are:

  • PeakUtils
  • Opencv-python
  • PyQt5
  • PyQt5-stubs

You can easly use CardiAP as Python library by installing it like follows:

pip install

Or simply download the executable file for runnig it as a local software.
