ABOUT

Here we present CardIAP, an open-source software for analyzing series of calcium handling phenomena from confocal microscopy images. CardIAP is a Python based tool, which allows users to easily work with a single file or a pool of images, and obtain representative amplitude and kinetics data, and it’s statistical analysis.

CardIAP allows users to elucidate and distinguish calcium pattern disorders by extracting and calculating whole cell and local transient´s parameters such us relative amplitude, time to peak, tau decay, time to half amplitude. Lastly, our software can easily detect discordancy, alternancy and disynchrony phenomena with a sensitivity of 100 % and a significance of 95%, meaning to be a robust and flexible tool.

Quick start using CardiAP

CardiAP is an open source tool, enterely developed in Python. The dependences used are:

  • PeakUtils
  • Opencv-python
  • PyQt5
  • PyQt5-stubs

pip install
